React Router v7 — Product Catalog

A modern multi-page Product Catalog application built with React and React Router v7.

🔹 Features

Nested routing with <Outlet />

Dynamic routes using :id

URL parameters with useParams()

Programmatic navigation using useNavigate()

Active links with NavLink

404 Not Found page

Responsive modern UI design

🔹 Routes

/ — Home

/products — Product List

/products/:id — Product Details

* — Not Found

🔹 Tech Stack

React • React Router v7 • Vite • CSS
